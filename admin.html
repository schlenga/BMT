🔐 Non-negotiables
  • Admin uses the same Worker as users — no separate API key, no alternate path.
  • Admin session = SSO + 2FA + CIDR + 15-min JWT. Expired = logged out mid-task.
  • Impersonation always requires operator consent via SMS code (their identity anchor) + is recorded.
  • Prompt bodies are never readable — not even by founders. Only meta.
  • Every write above has an audit row. If it's not in the log, it didn't happen.
📦 What we skip in v0
  • SAML / SCIM — SSO-only team is <10 people. Revisit at 15.
  • Per-tenant regional sharding — one EU region for now.
  • Customer-managed keys (BYOK for Claude) — enterprise-only, year 2.
  • Multi-region replica for admin reads — replica in same region is fine until 10k tenants.
  • Webhooks out to tenant systems — not our job in v0.
Design contract, one line: anything an admin can do, an operator could one day audit — because it happened to their business. That's why every row is logged, every suspend has a reason, every impersonation has their consent.
